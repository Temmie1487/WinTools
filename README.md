# WinTools

WinTools 是一个由Temmie开发的图一乐Windows系统管理工具，采用 WPF + WPF-UI 开发，具有 Fluent Design 风格和 Mica 背景效果。

## 简介

WinTools 旨在为用户提供一个界面美观、功能强大（并非）的 Windows 系统管理工具。无论是查看系统信息，管理进程服务，还是监控系统性能，WinTools 都能帮助你轻松（并非）完成。

## 功能特点

### 仪表盘
系统概览信息，包含计算机名、操作系统版本、当前用户、运行时间等实时数据。支持自定义快捷磁贴，可以快速导航到常用功能。

### 进程管理
查看所有运行中的进程，支持按名称搜索。可以结束进程、查看进程文件位置。需要管理员权限执行敏感操作。

### 服务管理
管理系统服务，查看服务状态，支持启动和停止服务。需要管理员权限。

### 性能监视
实时监控系统性能，包括 CPU 使用率、内存使用情况、磁盘读写速度、网络流量等。

### 磁盘管理
查看磁盘分区详细信息，包括盘符、文件系统类型、总容量和可用空间。

### 网络管理
管理网络适配器，支持启用/禁用适配器、刷新 DNS 缓存、释放/更新 IP 地址。

### 应用管理
查看已安装的应用程序列表，支持卸载应用、打开应用文件位置。

### 设备管理
查看计算机硬件设备信息，包括设备名称、设备类型、运行状态和驱动程序信息。

### 启动项管理
管理开机启动项，可以查看和禁用启动程序。

### 环境变量
查看和管理系统环境变量，包括用户变量和系统变量。支持添加、编辑、删除变量。

### 用户管理
管理 Windows 用户账户，查看用户信息。

### 电源管理
管理电源计划，查看电池状态，支持快速切换电源计划。

## 技术栈

- **.NET** - 跨平台 .NET 运行时
- **WPF** - Windows Presentation Foundation
- **WPF-UI** - Fluent Design UI 库
- **CommunityToolkit.Mvvm** - MVVM 框架
- **Serilog** - 日志框架

## 系统要求

- Windows 10/11
- .NET Runtime

## 构建

```bash
git clone https://github.com/Temmie1487/WinTools.git
cd WinTools
dotnet build
```
